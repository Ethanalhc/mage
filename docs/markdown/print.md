# `print` Statements
The print statement outputs text to the console, this is the simplest form of displaying information in mage.
## Syntax
```
print <VALUE>;
```

`<VALUE>` can be a string, integer, or variable.

---
## Examples:

```
>> print "Hello World!";
>> print "Goodbye World!";

Hello World!
Goodbye World!
```
---
```
>> print 0;
>> print 34;

0
34
```
---
```
>> let x = "Hi!";
>> print x;

"Hi!"
```
