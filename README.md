# Mage-lang
## A mini interpreter!
Mage is a small interpreted language built for simple syntax with only one rule: No, Brackets, Allowed!

* Think outside the **[ BOX ]**
* Beginner Friendly Syntax - Easy to read and learn
* Interactive REPL - write and run Mage instantly.

# How to install mage?
1. Install python: https://www.python.org
2. Download this folder as a ZIP
3. Extract this folder to a memorable place
4. Add python to your path
5. Open the folder in command line
6. Open the shell environment: 

```python src/shell.py```
